Can you prevent multiple votes (ballot stuffing)?

Well, if you give each respondent a username and password and have them login, you can limit multiple voting. If you don't force respondents to log in, it becomes more difficult (e.g., you could analyze the data by IP.)

Can phpESP support large numbers of respondents?

Yes, large surveys (100 questions) and large respondent groups (4,000+) can be handled without a problem. Throughput issues are generally a function of how fast is your web server, how fast is the mysql server, and what bandwidth access is there to your servers.

I'm running Windows and keep getting a blank screen-Help!

If is is not a pathing problem in phpESP.ini, then copy gnu_gettext.dll and php_gettext.dll (both usually found in the PHP install directory) into the system director of windows. For Win95/98 the system directory is something like c:/winroot/system and on WinNT/2000/XP the system directory is something like c:/winroot/system32.

After you have copied these dll's to the proper place, edit your php.ini file and comment out the line "extension=php_gettext.dll".

Last thing you do is stop and restart your server.
